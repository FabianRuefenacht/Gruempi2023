Das File "TeamsGruemi.json" enthält alle angemeldeten Teams.

Im File "gruppen.json" wird die Rangliste der Gruppe geführt. Dazu sind die Anzahl Punkte und die Tordifferenz jedes Spieles relevant.


Im File "Zeitplan.json" ist der Spielplan mit den Startzeiten und den spielenden Teams (sowie der Spielstand) entscheidend.
In diesem File wird der Spielstand geführt. Der Platzhalter dafür ist -1:-1. (Beispiel: "Endresultat": "-1:-1"). Bei diesem Wert wird der Spielstand auf der Website nicht dargestellt.


Bitte die beiden Files mit den Teamnamen und Spielzeiten ergänzen.


Der Link zum Spielplan / Rangliste: https://gruempi2023.sourcelab.ch/live
